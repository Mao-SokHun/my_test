# Coding conventions — quick reference

Full guide (team standard): **[`CONVENTION_GUIDE.md`](./CONVENTION_GUIDE.md)**

| Topic | Rule |
|-------|------|
| Files | lowercase, hyphens: `mentors-controller.js` |
| Variables / functions | camelCase; booleans: `isLoggedIn` |
| Classes / models | PascalCase: `Mentor` |
| Large comment | `// ============= Start name =============` + `// ....` |
| Small comment | `// Start name` … `// End name` + dots |
| Mentor folders | `controllers/mentor_system/`, `middleware/mentor_system/`, … |
| Verify paths | `cd backend_rokkru && npm run check:mentor-paths` |

Mentor tasks: [`MENTOR_SYSTEM_TASKS.md`](./MENTOR_SYSTEM_TASKS.md)
